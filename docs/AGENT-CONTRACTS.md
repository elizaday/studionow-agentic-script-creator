# Agent Contracts

Every agent returns JSON. Markdown belongs inside JSON fields only when the artifact is a script or notes deliverable.

## Diagnoser

Locks:

- format
- placement
- audience
- required understanding
- required feeling
- required action
- runtime
- approval reality
- existing assets
- opening tension
- closing move
- end feeling
- assumptions
- risks

## Miner

Extracts:

- human tension
- metrics
- strategic frameworks
- brand language
- asset notes
- clearance flags
- usable ammunition

## Strategist

Decides:

- whether directions are needed
- recommended direction
- concept engine
- story arc

If a human selected a direction, downstream stages must receive that selected direction as the recommended direction.

## Producer

Builds:

- title
- runtime
- tone
- visual motif
- structure
- opening move
- closing move
- production notes
- VO word budget

## Writer

Writes:

- metadata
- VO word count
- three-column script markdown

## Runtime Editor

Checks:

- VO density
- runtime realism
- weak lines
- revised markdown if needed

## Critic

Gates:

- generic language
- weak opening
- missing motif
- cold transitions
- unproduceable visuals
- runtime dishonesty
- claim risk

## Formatter

Delivers:

- final markdown
- producer notes
- delivery checklist

## Example Memory

Retrieves:

- relevant paired examples
- relevant script-only taste references
- teaching points
- source tags
- relevance scores

Examples guide taste and structure. They are not copied.

## Human Feedback

Captures:

- rating
- verdict
- reviewer comment
- suggested fix
- gold-candidate signal

Feedback is a learning signal. It should become training memory only after a human-approved final script is attached.
